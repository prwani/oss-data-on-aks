# Workload template

Use this template when onboarding the next blueprint.

## Minimum file set

- `docs/portal-deployment.md`
- `docs/az-cli-deployment.md`
- `infra/terraform/main.tf`
- `infra/bicep/main.bicep`
- workload `README.md`

## Minimum expectations

- reference the shared AKS AVM baseline
- document portal and CLI user journeys
- define workload-specific prerequisites
- capture operational and security considerations
- add a matching blog package when the workload is ready for external storytelling
