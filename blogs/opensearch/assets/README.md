# OpenSearch blog assets

Store final screenshots, diagrams, and cropped images here before publication.

Suggested assets:

- architecture diagram
- cluster creation screenshots
- OpenSearch health screenshots
- Dashboards access screenshots

Included source-based assets:

- `opensearch-cluster.png` - copied from OpenSearch documentation at https://docs.opensearch.org/latest/images/cluster.png
- `opensearch-cluster-replicas.png` - copied from OpenSearch documentation at https://docs.opensearch.org/latest/images/intro/cluster-replicas.png

The two official OpenSearch diagrams above come from the public `opensearch-project/documentation-website` repository, which is licensed under the Apache License 2.0.

Custom asset:

- `opensearch-on-aks-combined-architecture.svg` - combines the OpenSearch logical architecture with this repository's AKS deployment pattern
- `opensearch-on-aks-secure-architecture.svg` - shows the secure portal deployment with a private AKS API, VNet-integrated deployment script, internal Dashboards, keyless Blob snapshots, and encrypted persistent disks
